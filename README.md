# XXX - 优化版案例爬虫

这是一个基于 Python 开发的自动化案例抓取工具。本项目采用了超保守的延时策略和代理轮换机制，旨在在高效获取数据的同时，最大限度地保护目标服务器的稳定性。

## 🌟 核心功能
* **断点续爬**：自动保存检查点，支持从异常中断处恢复。
* **分批存储**：支持定时保存为 Excel 和 CSV 格式。
* **安全策略**：内置随机延时、多级重试以及自动休息模式。
* **代理支持**：完美适配快代理等私密代理 API。

## 🛠️ 环境准备
运行本项目需要以下 Python 库：
\`\`\`bash
pip install requests beautifulsoup4 pandas openpyxl
\`\`\`

## 🚀 快速开始
1. 克隆本项目到本地。
2. 打开脚本文件，将其中的 `XXX` 占位符替换为你真实的：
   - `BASE_URL` & `LIST_URL`
   - `HEADERS` 中的浏览器指纹
   - 代理 API 的 `USERNAME` 和 `PASSWORD`
3. 运行程序：
\`\`\`bash
python crawler_optimized.py
\`\`\`
